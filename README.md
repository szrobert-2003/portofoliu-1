# Landing Page Webdesign

Proiect Next.js pentru crearea de site-uri web moderne.

## Tehnologii

- **Next.js 14** - Framework React pentru producție
- **TypeScript** - Tipuri statice pentru JavaScript
- **Tailwind CSS** - Framework CSS utility-first
- **ESLint** - Linter pentru cod curat
- **Framer Motion** - Biblioteca pentru animații fluide
- **Lucide React** - Iconuri moderne și personalizabile

## Instalare

Toate dependențele sunt deja instalate! Dacă ai nevoie să reinstalezi, rulează:

```bash
npm install
```

## Rulare

Pentru a porni serverul de dezvoltare:

```bash
npm run dev
```

Apoi deschide [http://localhost:3000](http://localhost:3000) în browser.

## Comenzi disponibile

- `npm run dev` - Pornește serverul de dezvoltare
- `npm run build` - Construiește aplicația pentru producție
- `npm run start` - Pornește serverul de producție
- `npm run lint` - Rulează ESLint pentru verificarea codului

## Structura proiectului

```
├── src/
│   └── app/
│       ├── layout.tsx    # Layout-ul principal
│       ├── page.tsx      # Pagina principală
│       └── globals.css   # Stiluri globale
├── package.json
├── tsconfig.json
├── tailwind.config.ts
└── next.config.js
```

## Începe să codezi!

Editează `src/app/page.tsx` pentru a începe să construiești landing page-ul tău!
